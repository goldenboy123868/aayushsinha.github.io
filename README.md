## âœï¸ Customization Guide

### 1. Personal Information
In `index.html`, replace:
- `Your Name` (appears in multiple places)
- `your.email@example.com`
- LinkedIn URL: `https://linkedin.com/in/yourprofile`
- GitHub URL: `https://github.com/yourusername`

### 2. Projects
For each project card:
- Replace project title
- Update problem, role, approach, and results
- Change tags to match your tech stack
- Add GitHub/demo links
- Replace image placeholders with actual images

To add an image:
```html
<!-- Replace this: -->
<div class="project-image">
    <span>Project Image/Demo</span>
</div>

<!-- With this: -->
<div class="project-image">
    <img src="images/your-project.jpg" alt="Project Name">
</div>
```

### 3. Skills
Edit the skill categories in the Skills section to match your expertise. Feel free to add or remove categories as needed.

### 4. About Section
- Write your bio (3-4 paragraphs)
- Add your interests
- Replace profile photo placeholder with your image

### 5. Resume
- Add your resume PDF to `files/resume.pdf`
- The download button will automatically work

### 6. Colors (Optional)
To change the color scheme, edit `css/styles.css`:
- Primary color: `#0066cc` (change all instances)
- Text color: `#333`
- Background: `#f8f9fa`
