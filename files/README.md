# Files Folder

Place your downloadable files here:

## Required Files:
- `resume.pdf` - Your resume/CV in PDF format

## Guidelines:
- Keep PDF under 2MB
- Name it exactly `resume.pdf` (or update the link in index.html)
- Make sure the PDF is up-to-date and matches your portfolio content
- Test the download link after deploying

## Tips:
- Use a professional resume template
- Include the same projects featured on your portfolio
- Keep it to 1-2 pages maximum
- Export as PDF to preserve formatting
