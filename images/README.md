# Images Folder

Place your images here:

## Required Images:
- `profile.jpg` or `profile.png` - Your profile photo (recommended: 500x500px)
- `project1.jpg` - First project screenshot/demo
- `project2.jpg` - Second project screenshot/demo
- `project3.jpg` - Third project screenshot/demo

## Image Guidelines:
- Format: JPG or PNG
- Size: Keep under 500KB each for faster loading
- Dimensions: 1200x800px recommended for project images
- Profile photo: Square aspect ratio (500x500px or 1000x1000px)

## Tips:
- Use descriptive names (e.g., `arduino-robot.jpg` instead of `IMG_1234.jpg`)
- Optimize images before uploading using tools like TinyPNG or Squoosh
- Use JPG for photos, PNG for screenshots with transparency
